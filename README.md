# even-tools

A collection of useful web tools.

## Tools

- **Color Calculator** - Color conversion, contrast analysis, harmonies, and blindness simulation
- **Floor Plan Generator** - Create and export floor plans with drag-and-drop components
- **Habit Tracker** - Track daily habits with calendar views and charts
- **Regex Sandbox** - Test and visualize regular expressions with presets
- **Unit Converter** - Convert between various units with global search
- **Visual Page Builder** - Build web pages visually with drag-and-drop components

## Tech Stack

React + TypeScript + Vite + Zustand + SCSS

## Development

```bash
yarn install
yarn dev
```

## Build

```bash
yarn build
```

## Deploy

